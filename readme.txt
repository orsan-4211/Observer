○1.本システムについて

　　この度は「Observer」を手に取っていただきありがとうございます。

　　本システムはトラフィックの可視化、および監視を目的としたソフトとなっております。

○2.インストール方法

　1.WinPcap 4.1.2以上をインストールする
　2.本システムを解凍し、任意のフォルダ内に配置する
　
　　以上の手順でインストールが完了いたします。「Observer.exe」ファイルを実行してお使いください。

○3.アンインストール方法

　　本システムはインストーラは使用しておりませんので、アンインストーラーは付属しておりません。
　　そのままゴミ箱に入れて削除していただいて結構です。

○4.動作環境
　OS　　　　　　 : Windows 8.1, 10 (デスクトップモードのみ)
　CPU 　　　　　 : Intel® Core™ i5 以上 (i7 シリーズ以上推奨)
　メモリ　　　　 : 4GB 以上(8GB 以上推奨)
　ハードディスク : 約 4MB 程度必要
　グラフィック　 : DirectX 11の動作するビデオカード
　　　　　　　　   Intel® HD Graphics 等のオンボードグラフィックの場合、
　　　　　　　　   正常に動作しない場合があります。
　ディスプレイ　 : 1600 × 900 以上( 1980 × 1080 以上推奨)
　　　　　　　　   これより小さい場合、項目が隠れて利用できなくなる等、
　　　　　　　　   正常に使用できない場合があります。

○5.操作方法

　マウスボタン左　：項目の選択
　マウスボタン中央：画面のスライド
　マウスホイール　：画面の拡大縮小
　Ｒキー　　　　　：画面の移動のリセット
　Ｗキー　　　　　：マーカーのフリー移動のリセット

○6.コンフィグファイル

　　ネットワーク構成図の設定を行うことができます。
　　以下の記述を行うことで設定できます。
　　
「(IPアドレス)[スペース]-[スペース](親IPアドレス)」

　　親IPアドレスに「-1」を指定することで親IPアドレスを指定せずにIPアドレスを設定することができます。
　　スペースの部分はタブでも代用可能です。
　　ファイルの最後は必ず空白行になるように記述してください。
